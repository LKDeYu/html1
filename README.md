# NAMRANTA Portfolio

吴志宏的个人作品集网站，用于展示 AI 学习方向、项目经历、学习笔记、留言系统和阿里云部署实践。

## Tech Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS
- GSAP + ScrollTrigger
- Lenis
- Motion
- SQLite planned

## Local Development

```bash
npm install
npm run dev
```

Open <http://127.0.0.1:3000>.

## Scripts

```bash
npm run lint
npm run build
```

## Project Direction

- Visual landing page with iOS-style glassmorphism and liquid gradients
- Scroll-driven cinematic portfolio sections
- Blog CMS with Markdown, categories, tags, search, drafts, pinned posts, and views
- Guestbook with admin approval
- Final deployment target: Alibaba Cloud ECS + Ubuntu + Nginx + PM2
